# Evaluacion_U4_Markdown_APSL
## Proceso de Configuración

1. Abrí GitHub Desktop en mi computadora.
2. Hice clic en "Create a New Repository".
3. Escribí el nombre del repositorio como "Evaluacion-Markdown-NombreApellido".
4. Seleccioné la ubicación donde guardar el reposorio en mi computadora.
5. Inicialicé el repositorio con un archivo README.md.
6. Hice clic en "Create Repository".
7. Luego presioné el botón "Publish Repository".
8. Me aseguré de que el repositorio fuera público.
9. Finalmente verifiqué en GitHub que el repositorio estuviera visible en la web.
